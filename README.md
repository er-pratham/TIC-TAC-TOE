
# TIC TAC TOE

A Cross Zero Web game based on frontend tools and enables user to two users play at a time.

## Features

- Two Players Can Play At a time
- Match Tie Logic   
- Match Win Logic
- Turn Sounds


## Tech Stack

**Client:** React JS, Material UI, HTML, JSX, CSS, Javascript etc 

**Server:** Github Pages.
